How I did it:

  1. Installed Samba onto my Dell N7010 via Linux Mint Xfce terminal
  2. Created a shared directory so that my entire household can access the folder
  3. Configured samba by specifying the folder path, permissions, and setting
    mask/directory mask
  4. Created a password and tied it to my already existing username in order to
    limit access to the directory
